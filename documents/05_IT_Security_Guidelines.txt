IT Security Guidelines for ProjectX

Access Control:

- All users must authenticate using company SSO
- Admin access requires multi-factor authentication (MFA)
- API keys must never be stored in plaintext

Password Policy:

- Minimum length: 12 characters
- Must include uppercase, lowercase, number, and symbol

Data Protection:

- Sensitive documents must be encrypted at rest
- All network traffic must use HTTPS
- Internal services must use TLS

Prohibited actions:

- Sharing credentials
- Exporting sensitive internal documents externally
- Uploading customer PII without authorization

Incident Reporting:

All security incidents must be reported within 1 hour to:

security@company.internal
