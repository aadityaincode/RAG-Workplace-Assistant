Deployment Guide for ProjectX

Prerequisites:

- Docker installed
- Kubernetes cluster access
- kubectl configured

Steps:

1. Build Docker image

docker build -t projectx:latest .

2. Push image to registry

docker push registry.internal/projectx:latest

3. Apply Kubernetes deployment

kubectl apply -f deployment.yaml

4. Verify pods

kubectl get pods

Expected output:

projectx-api running
projectx-db running
projectx-worker running

Production deployments require approval from DevOps team.
